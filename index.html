quicksort

发表于 2023-04-12 分类于 -code

快速排序C++版：
比较简单，所以没有注释：

需要注意的一些点就是：

  1. 声明数组的长度的时候是怎么声明的
  2. 快排函数以j为边界的时候怎么写的
  3. 核心思想是分治
#include<iostream>
using namespace std;
void quicksort(int q[], int l, int r);
const int N = 1000010;

int main(){
    int n;
    cin>>n;
    int q[N];
    for (int i=0;i<n ; i++){
        cin>>q[i];
    }
    quicksort(q,0,n-1);

    for (int j=0;j<n-1;j++){
        cout<<q[j]<<" ";
    }
    cout<<q[n-1];
    return 0;

    }
    void quicksort(int q[],int l, int r){
    	if( l>= r){
        return;
    }

    int i,j,tmp;
    i = l-1;j = r+1;
    int x = q[(l+r)>>1];
    while(i< j){
    do i++; while(q[i]< x);
    do j--; while(q[j] >x);
    if (i<j){
        tmp = q[i];
        q[i]  = q[j];
        q[j] = tmp;

    }
    }
    quicksort(q,l,j);
    quicksort(q,j+1,r);
}

单周期原理

指令

首先需要知道Y-86的指令是如何编码的，是一个两位的数字，第一位是操作码，第二位是功能码，三四位时寄存器的号，后面八个字节是需要使用的立即数的值（用来基地址寻址或者是立即数跳转），具体的详见，Y-86指令速查，里面有详细地介绍指令。

各个工作阶段

分为六个阶段：

  • 取指
  • 译码
  • 执行
  • 访存
  • 写回
  • 更新PC

def execuate():  ###执行
globals()
global icode, ifun, rB, rA, valE, valC, valA, valB, valP, valM, Register, Opcode_trans, \

    if instruction == 'addq':

    elif instruction == 'subq':

    elif instruction == 'xorq':
        valE = valA ^valB
    ZF = 1 if valE == 0 else 0
    SF = 1 if valE < 0 else 0
    OF = ((valA<0) == (valB<0)) & ((valE<0) != (valA<0))

    valE = valA

    valE = valB + valC

    Cnd = 1
elif instruction=='jle':
    Cnd = (SF^OF)|ZF
elif instruction == 'je':
    Cnd = ZF
elif instruction == 'jl':
    Cnd =SF^OF
elif instruction == 'jne':
    Cnd = 1-ZF
elif instruction =='jge':
    Cnd = ~(SF^OF)
elif instruction == 'jg':
    Cnd = ~(SF^OF) & ~ZF
elif instruction == 'call':
    valE = valB - 8
elif instruction =='ret':
    valE = valB + 8

return True

自制markdown语法说明

发表于 2023-04-01 更新于 2023-04-03 分类于 notes

段落：

  1. “—-” ： 分割线如上
  2. “两个空格加换行” ： 换行
  3. “中间隔一个空行” ：创建段落

字体

  1. 粗体 ： 四个星号中间包围
  2. 斜体 ： 两个星号中间
  3. 粗体加斜体 ： 三个星号中间包围着

引用

  1. 引用块 ： “>”在大于号后面添加内容即可
  2. 多段落引用： “>”在段落与段落之间的空行上面也加入“>”
  3. 引用格式：引用块中同样遵循Markdown格式
  4. 文内链接：***作者的网站***,【】（）中括号里面是文本内容，小括号是链接地址 ，给出链接地址之后可以加“”,在双引号里编辑提示。可以被星号强调以及斜体

文内格式

  1. 列表 ： 数字列表直接输入数字“1.”，并在小数点后加空格

  2. 无序列表：用“-”代替“1.”

  3. 在列表中保留元素：加入一个tab缩进

    • 嵌套列表：加入tab缩进之后继续使用“1.”或者“-”

  4. 代码块：直接缩进四个空格，如果在列表中，缩进8个空格,与正文中间留空行 

     #include<stdio.h>  
 int  main(void)  {  
     printf("hello,world!");
     return 0;
 }

  5. 单词或短语转为代码：“code time”,用反引号标注起来

  6. 图片：要添加图像，请添加感叹号（!），然后在中括号中添加替代文本，并在小括号中添加图像资源的路径

  7. 转义字符： 显示Markdown语法所占用的原有的字符，“/”后面添加即可，”\ *” = *
